# 🎄 圣诞树 - 节日魔法 (Christmas Tree Magic)

一个基于 **Three.js** 和 **MediaPipe** 构建的 3D 互动圣诞体验。通过计算机视觉技术，让你用手势控制流光溢彩的粒子圣诞树。

## ✨ 核心亮点

- **沉浸式 3D 场景**：由物理粒子组成的金色圣诞树，配合电影级的辉光（Bloom）特效。
- **AI 手势识别**：无需鼠标键盘，通过摄像头捕捉手部动作与场景互动。
- **照片回忆墙**：支持上传你自己的照片，它们会镶嵌在金色相框中，成为圣诞树的一部分。

## ✋ AI 手势交互指南

请允许浏览器访问摄像头权限，并站在距离摄像头约 50cm-1m 处。

| 手势 | 动作说明 | 魔法效果 |
| :--- | :--- | :--- |
| **握拳** ✊ | 五指握紧 | **召唤圣诞树**：散落的粒子会重新汇聚成旋转的圣诞树。 |
| **张开手掌** ✋ | 五指张开 | **漫天星散**：圣诞树炸裂，化作漫天飞舞的流星与尘埃。 |
| **捏合手指** 👌 | 拇指与食指捏合 | **聚焦回忆**：随机抓取一张照片（或你上传的照片）拉近到眼前展示。 |
| **手掌移动** 👋 | 上下左右挥动 | **控制视角**：场景会跟随你的手掌位置进行 3D 旋转。 |

> 💡 **提示**：如果识别不灵敏，请确保环境光线充足，并露出完整的手掌。

## 🚀 如何运行

1.  直接在浏览器中打开 `index.html` 即可体验。
2.  点击 **"ADD MEMORIES"** 按钮可以上传一张照片。
3.  按键盘 **'H'** 键可以隐藏所有 UI 界面，享受纯净视觉体验。

> **⚠️ 注意**：由于浏览器的安全策略，摄像头访问必须在 **HTTPS** 协议或 **localhost** 环境下才能工作。推荐部署到 **GitHub Pages**。

## 🛠️ 技术栈

- **Three.js (WebGL)**：3D 渲染与粒子系统
- **Google MediaPipe**：实时手部关键点检测
- **Vanilla ES6 JavaScript**：原生模块化开发，单文件组件
